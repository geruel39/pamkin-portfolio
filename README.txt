# 👥 Team Portfolio

A modern, responsive team portfolio built with **React**, **React Router**, and **TailwindCSS**, powered by **Vite**. This site showcases team members, their roles, and project highlights in a sleek, mobile-friendly design.

## ⚙️ Installation & Setup

### Prerequisites

- Node.js >= 16
- npm or Yarn

### Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/yourusername/team-portfolio.git
cd team-portfolio
npm install

Run the development server:
npm run dev
Open http://localhost:5173 to view the app

🧱 Project Structure
