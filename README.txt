# 👥 Team Portfolio

A modern, responsive team portfolio built with **React**, **React Router**, and **TailwindCSS**, powered by **Vite**. This site showcases team members, their roles, and project highlights in a sleek, mobile-friendly design.

---

## 🚀 Live Demo

[🔗 View the Live Site](https://pamkin-team-portfolio.vercel.app/)


🧱 Project Structure
portfolio/
├── public/         	  # Images, svgs
├── src/		  # Source Code
│   ├── components/       # Reusable components (Navbar, MobileMenu, Button)            # Route-specific pages (Home, About, Contact)
│   │   ├── pages/	  # Route-specific pages (Home, About, Contact)
│   ├── App.jsx           # App entry with routing
│   └── main.jsx          # ReactDOM render entry point
├── index.html		  # Entry point
├── vite.config.js        # Vite config with Tailwind plugin
├── package.json          # Project metadata and scripts
└── README.txt            # You're here!


🎨 Technologies Used

-⚛️ React

-🧭 React Router

🧩 Features

-Responsive layout

-Smooth client-side routing with React Router

-Dynamic team member rendering from static data

-Utility-first CSS with Tailwind

-Clean and modular component structure
